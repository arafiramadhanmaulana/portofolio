Tugas Akhir Machine Learning IPCA

Pengaruh Cumulative Explained Variance pada Incremental Principal Component Analysis terhadap Klasifikasi Naïve Bayes 16S rRNA Berbasis k-mer

Proyek Tugas Akhir ini bertujuan untuk menganalisis pengaruh variasi target Cumulative Explained Variance (CEV) pada metode Incremental Principal Component Analysis (IPCA) terhadap performa klasifikasi algoritma Naïve Bayes pada taksonomi bakteri tingkat Phylum. Dataset sekuens gen 16S rRNA dari basis data SILVA dikonversi menjadi representasi numerik menggunakan metode ekstraksi fitur k-mer. Reduksi dimensi menggunakan IPCA diterapkan untuk mencegah fenomena curse of dimensionality. Hasil eksperimen menunjukkan bahwa dimensi fitur berhasil direduksi sebesar 97,12% dengan Akurasi tertinggi sebesar 96,89% dan nilai AUC 0,9994.
